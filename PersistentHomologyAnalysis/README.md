# Running the Analysis

This directory contains the Jupyter notebook `analysis.ipynb` that reproduces the persistent homology computations from the paper.

## Prerequisites

### Python Version
- Python >= 3.13

### Required Packages

The following Python packages are required to run the notebook:

- [Correa](https://correa.yossi.eu) - for computing persistence diagrams
- plotly - for interactive visualizations
- pandas - for data manipulation
- scikit-learn - for clustering analysis
- numpy - for numerical computations
- matplotlib - for plotting
- tifffile - for reading microscopy images
- kaleido - for static image export from plotly
